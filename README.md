# Deborah Salako Portfolio

Static portfolio website generated to match the layout and content of the sample site.

## Files

- `index.html` — main landing page
- `styles.css` — page styles
- `script.js` — minimal form behavior

## Deploy

This site is ready to deploy as a static website on any cloud hosting service such as Vercel, Netlify, GitHub Pages, or Azure Static Web Apps.

## Usage

1. Open `index.html` in a browser.
2. Customize text, links, and content as needed.
3. Replace placeholder images with your own assets.
